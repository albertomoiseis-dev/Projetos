# Calculadora Aritmética Básica

## Descrição do Projeto

Este projeto é uma calculadora aritmética básica desenvolvida com HTML, CSS e JavaScript. A calculadora permite realizar as seguintes operações:

- Adição (+)
- Subtração (-)
- Multiplicação (\*)
- Divisão (/)

## Tecnologias Utilizadas

- **HTML**: Define a estrutura da calculadora, com os botões numéricos, de operações e o display.
- **CSS**: Responsável pelo layout e estilo da calculadora, incluindo o design dos botões e do display.
- **JavaScript**: Implementa a lógica da calculadora, capturando as entradas dos botões, realizando os cálculos e atualizando o display.

## Estrutura do Código

O código está organizado de forma modular, com funções específicas para cada tarefa, tais como:

- Atualizar o display
- Limpar a calculadora
- Remover o último dígito
- Realizar as operações aritméticas

Além disso, o código está comentado em português e os nomes das variáveis e funções também estão em português para facilitar o entendimento e a manutenção.

## Funcionalidades

- **Adição**: Soma dois números.
- **Subtração**: Subtrai um número do outro.
- **Multiplicação**: Multiplica dois números.
- **Divisão**: Divide um número pelo outro.

## Como Utilizar

1. Clone o repositório para sua máquina local.
2. Abra o arquivo `index.html` em seu navegador.
3. Utilize os botões para realizar cálculos aritméticos básicos.
